What it actually does

No speed limit

Every working-speed clamp in the game funnels through one function — Vehicle.getSpeedLimit. This mod overrides it to return "no limit." Cultivators, seeders, sprayers, balers, cutters: all of them work at whatever speed you're doing. Damage-based speed reduction dies with it. Factory top speed gets a 2x multiplier and cruise control won't re-cap you until 200 km/h.

Max capacity

A pattern table matches vehicles by their config path and rewrites fill unit capacity on load. Works on base game vehicles and mod vehicles the same way. Fill mass is ignored by default — because 940,000 liters of wheat is roughly 730 tons and the physics engine has opinions about that.

Max horsepower

Torque gets scaled at the source — an override on VehicleMotor.getTorqueCurveValue — so it works no matter how GIANTS stores the curve internally. Rigs are matched by config-path substring with explicit per-vehicle overrides — John Deere 9R and 9RX, Case IH Steiger, CLAAS XERION 12, Volvo VNX 300, Mack Anthem — and adding another is one line in the table. Heads up: the shop page still shows factory horsepower because it reads cached store data. The tractor pulls 1156. The brochure lies.

Install

  1. Drop FS25_MichalsInGameMods.zip into your mods folder — Documents\My Games\FarmingSimulator2025\mods\ (or wherever your custom mod directory points). Don't extract it. Don't rename it — the filename is the mod name and GIANTS rejects dots and version suffixes.
  2. Enable it when loading your savegame.
  3. Optional: grep log.txt for MIGM: to see exactly what got patched.

Make it yours

Everything lives at the top of scripts/MichalsInGameMods.lua inside the zip. Change values, re-zip, done.

MIGM.TOP_SPEED_MULTIPLIER = 2.0   -- physics gets comedic above ~3.0
MIGM.CRUISE_MAX_KMH       = 200

MIGM.CAPACITY = {
    ["pacesetter"] = 940000,   -- match by config-path substring
    ["tdk301"]     = 60000,    -- add your own trailers here
}
MIGM.IGNORE_FILL_MASS = true    -- false = realistic 730-ton comedy

MIGM.BIG_TRACTOR_HP = 1156
MIGM.BIG_TRACTOR_BRANDS = { JOHNDEERE=true, CASEIH=true, CLAAS=true, FENDT=true }

Not sure what a vehicle's config path looks like? Set MIGM.DEBUG = true, load in, and every vehicle prints its exact path to the log. Copy the distinctive part into the table. That's the whole workflow.

Known quirks — read before you complain

Singleplayer only. The mod declares multiplayer supported="false" and means it. The motor changes won't sync cleanly across clients.
Shop specs don't update. Store pages read cached data. Judge by the seat of your pants or a HUD mod, not the brochure.
Unload speed is unchanged. Tipping 940k liters at a sell point takes a while. That's the discharge rate, not the capacity.
Game patches can rename things. If a GIANTS update breaks a hook, the log will tell you exactly where. That's on purpose.
